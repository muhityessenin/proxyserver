# Proxy Server

## Установка и запуск

### Требования

- Docker
- Docker Compose

### Запуск сервера

1. Склонируйте репозиторий:

git clone <(https://github.com/muhityessenin/proxyserver.git)>
cd <your-repo-name>

2. Постройте и запустите контейнеры:
make build
make up

Сервер будет доступен по адресу http://localhost:8080.
